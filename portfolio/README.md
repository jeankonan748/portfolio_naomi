# Resume - Free Next.js Resume Website Template
#### Preview

 - [Demo](https://themewagon.github.io/Resume-Nextjs/)


## Getting Started

Clone from GitHub 
```bash
git clone 
```

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
